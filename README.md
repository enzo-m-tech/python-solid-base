# 🐍 Python — Base Sólida

Este repositório foi criado com o objetivo de desenvolver uma base sólida em Python e lógica de programação.

Aqui estão organizados meus estudos, exercícios, anotações e desafios práticos enquanto evoluo na linguagem.

---

# 🎯 Objetivo

O foco deste repositório é:

- Fortalecer lógica de programação
- Aprender Python do básico ao avançado
- Criar consistência nos estudos
- Desenvolver organização de código
- Construir uma base sólida para projetos futuros

Os exercícios foram feitos com foco em:
- entender a lógica
